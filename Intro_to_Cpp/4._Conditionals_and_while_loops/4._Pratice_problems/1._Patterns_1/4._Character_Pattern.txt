====== 4. Character Pattern ======
Created Friday 18 October 2019

* Remember here to cast the type to char.
* Otherwise, we can store the value to be printed in a char variable before printing.
e.g 
ABCD
BCDE
CDEF
DEFG
val(i, j) = char('A' + i + j -1 )
